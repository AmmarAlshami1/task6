Simple To-Do List App
Overview
This is a simple to-do list application designed to help you keep track of tasks and manage your time effectively. It provides basic functionalities to add, remove, and mark tasks as completed.

Features
Add Tasks: Easily add new tasks to your to-do list.
Remove Tasks: Delete tasks that are no longer needed.
Mark Completed: Mark tasks as completed once you finish them.
Simple Interface: User-friendly interface for easy navigation.
Installation
Clone or download the repository.
Open the project in your preferred code editor.
Run index.html in your web browser.
Usage
Adding a Task: Simply type your task into the input field and press Enter to add it to the list.
Removing a Task: Click on the delete icon next to the task you want to remove.
Marking Completed: Click on the checkbox next to the task to mark it as completed.
